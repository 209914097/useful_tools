## 准备

- python 3.6<br>
- IDE：pycharm<br>
- python库：<br>

python库 | 描述
--|--
pyqt5  |
PyQt5-tools | 包含Qt designer软件，用于做界面
pyinstaller|用于打包生成exe文件

因为pip安装库很慢，所以用清华镜像安装,输入指令如下：

```pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 库名```
